# Script Todo

# Description
Ce script todo vous aide à gérer vos tâches quotidiennes. Il permet de créer, mettre à jour, supprimer, afficher et rechercher des tâches. Chaque tâche est identifiée par un identifiant unique et contient les informations suivantes : titre, description, localisation, date et heure d'échéance, et statut de complétion.



# Installation
1. Téléchargez le script todo et sauvegardez-le dans un fichier nommé todo.
2. Rendez le script exécutable avec la commande suivante :
    bash
    chmod +x todo
    

# Utilisation
Lancez le script avec la commande suivante :
bash
./todo

Menu principal :

Le menu principal offre les options suivantes :

Créer une tâche
Mettre à jour une tâche
Supprimer une tâche
Afficher toutes les informations d'une tâche
Lister les tâches d'un jour donné
Rechercher une tâche par titre
Quitter

exemple :
Choisissez une option : 1
Titre : Acheter du lait
Date et heure d'échéance (YYYY-MM-DD HH:MM:SS) : 2024-05-20 14:00:00
Description (facultatif) : Acheter du lait entier
Localisation (facultatif) : Supermarché
Complétion (true/false) : false

Choisissez une option : 2
Entrez l'ID de la tâche à mettre à jour : aaaaa
Titre : Acheter du lait et du pain
Date et heure d'échéance (YYYY-MM-DD HH:MM:SS) : 2024-05-20 15:00:00
Description (facultatif) : Acheter du lait entier et du pain complet
Localisation (facultatif) : Supermarché
Complétion (true/false) : false

Choisissez une option : 3
Entrez l'ID de la tâche à supprimer : aaaaa

Choisissez une option : 4
Entrez l'ID de la tâche à afficher : aaaaa

Choisissez une option : 5
Entrez la date (YYYY-MM-DD) pour lister les tâches : 2024-05-20

Choisissez une option : 6
Entrez le titre pour rechercher : lait

Choisissez une option : 7
[Le script se termine]


Voila ! vous pouvez utiliser ce script
